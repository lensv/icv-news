### 执行命令（在每日流程的阶段二中自动运行）
```

C:/Users/18351/.workbuddy/binaries/python/envs/default/Scripts/python.exe scripts/collect_public_accounts.py
```

### 采集流程
1. Playwright（隐身模式：随机UA/随机视口/隐藏webdriver/随机滚动）访问搜狗微信
2. 按10个关键词逐页检索，每词翻3页
3. 关键词：智能网联汽车、自动驾驶、Robotaxi、智能驾驶、车路云、L3级自动驾驶、城市NOA、组合驾驶辅助安全要求、智慧交通车路协同、无人驾驶商业化
4. 窗口过滤：只保留运行日前1天发布的文章
5. 二次跳转至 mp.weixin.qq.com 取真实URL + 正文摘要（300字）
6. 自动解析汇编类文章（日报/晨报/速览等）拆出子条目
7. 去重：已存在于 news_index.json 的标题跳过
8. 输出至 data/public_accounts.json

### 反爬策略
- 6组User-Agent随机切换（Chrome/Firefox/Safari）
- 关键词间8-15秒随机间隔
- 翻页间5-10秒随机间隔
- 每篇文章取正文3-6秒间隔
- 触发验证码时等待120秒后重试
- 每次请求前随机滚动页面模拟人类

## 五、异常处理
- 单个网站无法访问时跳过，继续处理下一个
- 即使部分网站抓取失败也要生成可用报告
- 无新增新闻时生成空报告注明"今日暂无新增智能网联汽车相关新闻"
- 公众号采集触发验证码时等待120秒后重试，持续失败则跳过该关键词

## 六、数据库存储（SQLite）

每次采集后，数据写入统一数据库 `data/icv_news.db`。

### 表1：articles（新闻文章主表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| title | TEXT | 文章标题 |
| source | TEXT | 来源名称（盖世汽车 / 深圳商报（公众号）） |
| source_type | TEXT | 来源类型：gov/media/enterprise/wechat |
| url | TEXT UNIQUE | **真实链接**（去重 key） |
| publish_date | DATE | 发布日期 |
| collected_at | DATETIME | **采集时间** |
| category | TEXT | 分类：政策法规/标准动态/投融资动态/技术动态/项目动态/行业动态 |
| summary | TEXT | AI生成摘要（300字连贯文字） |
| full_content | TEXT | **正文全文/前1000字**（防止原文链接失效后找不到） |
| is_wechat | INTEGER | **是否来自微信公众号**（0/1） |
| importance | INTEGER | 重要性评分 1-5（0=默认） |
| window | TEXT | 采集窗口（如 2026-07-12） |
| overview | TEXT | 一句话概览 |
| keywords | TEXT | 关键词标签（JSON数组） |

### 表2：collect_log（采集运行日志）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| run_date | DATE | 运行日期 |
| version | TEXT | 版本号（v1/v2…） |
| window | TEXT | 采集窗口 |
| total_articles | INTEGER | 本次采集文章数 |
| total_public_accounts | INTEGER | 其中公众号文章数 |
| categories | TEXT | 各分类计数（JSON） |
| source_summary | TEXT | 数据来源说明 |
| note | TEXT | 备注 |
| status | TEXT | 运行状态（ok/error） |
| created_at | DATETIME | 创建时间 |

### 建库命令
```
cd e:\trae_solo\自动化监测智能网联汽车新闻
python scripts/db_schema.py create     # 仅建空库
python scripts/db_schema.py migrate    # 建库+从旧 news.db 迁移历史数据
```

### 查询示例
```sql
-- 查某窗口的全部新闻
SELECT title, source, category FROM articles WHERE window = '2026-07-12';

-- 查某分类近30天的新闻数
SELECT category, COUNT(*) FROM articles
WHERE publish_date >= date('now', '-30 days')
GROUP BY category;

-- 查公众号来源的新闻
SELECT title, source FROM articles WHERE is_wechat = 1;

-- 查最近5次采集日志
SELECT run_date, version, total_articles, categories
FROM collect_log ORDER BY id DESC LIMIT 5;
```
