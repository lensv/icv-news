## 任务一：每日新闻监测

- **名称**：智能网联汽车每日新闻监测
- **Cron 表达式**：`0 9 * * 1-5`（工作日每天上午 9:00）
- **时区**：Asia/Shanghai

### Message 内容：

```
你是智能网联汽车新闻每日监测系统。请采集昨天和今天的智能网联汽车相关新闻，生成当日HTML简报。

## 二、WebSearch补充搜索（第二优先级）

搜索关键词（限定昨日+今天）：
1. "智能网联汽车" + (政策/标准/投融资/自动驾驶/车路协同/V2X/测试示范)
2. "高工智能汽车" 最新
3. "36氪 汽车融资" OR "36氪 自动驾驶"
4. "界面新闻 汽车" OR "澎湃新闻 智能网联"
5. "甲子光年 自动驾驶"
6. 企业动态：华为乾崑、小马智行、文远知行、萝卜快跑、小鹏智驾、蔚来NOP+、理想智驾、小米汽车、大疆车载、Momenta、地平线、黑芝麻
7. 国际动态：GRVA、UN R157、NHTSA automated vehicle、SAE automated driving
8. 英文： "connected automated vehicle" OR "intelligent connected vehicle"

## 三、处理步骤
1. 读取 e:\trae_solo\自动化监测智能网联汽车新闻\data\news_index.json 进行URL去重
2. 基于关键词分类为六大类：政策法规、标准动态、投融资动态、技术动态、项目动态、行业动态
   - 政策法规: 政策、法规、管理办法、实施细则、准入管理、数据安全、网络安全
   - 标准动态: 国标GB、标准立项、征求意见、标准发布、标准修订、团体标准、UN R157
   - 投融资动态: 融资、投资、并购、收购、IPO、上市、估值、产业基金、战略合作
   - 技术动态: 自动驾驶、L2-L5、ADAS、激光雷达、传感器融合、车路协同、V2X、大模型、端到端、OTA
   - 项目动态: 道路测试、示范应用、Robotaxi、试点城市、先导区、测试牌照、商业化运营
   - 行业动态: 行业报告、市场数据、渗透率、产业趋势、竞争格局、行业白皮书、排行榜、出海
3. AI为每条新闻生成一段连贯的文字摘要（不分点，概括核心事件、关键数据、影响意义）
4. 参考 e:\trae_solo\自动化监测智能网联汽车新闻\templates\daily_template.html 的设计风格生成HTML
5. 保存到 e:\trae_solo\自动化监测智能网联汽车新闻\reports\daily\YYYY-MM-DD.html
6. 将新增新闻追加到 news_index.json，更新 last_update.json

## 四、异常处理
- 单个网站无法访问时跳过，继续处理下一个
- 即使部分网站抓取失败也要生成可用报告
- 无新增新闻时生成空报告注明"今日暂无新增智能网联汽车相关新闻"
```

## 任务二：月度汇总报告

- **名称**：智能网联汽车月度分析报告
- **Cron 表达式**：`0 10 1 * *`（每月1日上午 10:00，生成上月报告）
- **时区**：Asia/Shanghai

### Message 内容：

```
你是智能网联汽车新闻月度报告生成系统。请汇总上个月的全部新闻数据，生成月度分析报告。

## 执行步骤
1. 读取 e:\trae_solo\自动化监测智能网联汽车新闻\data\news_index.json 中上个月的全部新闻数据
2. 统计各类别（政策法规/标准动态/投融资动态/技术动态/项目动态/行业动态）新闻数量
3. 统计来源分布（哪些机构/媒体贡献了新闻）
4. AI分析本月热点趋势、重要政策节点
5. 评选Top 10重要新闻，生成深度摘要（一段连贯的长段落，涵盖背景、核心内容、关键数据、行业影响）
6. 生成趋势分析文字（本月政策重点方向、标准制修订热点、投融资活跃领域、技术发展亮点、重大项目进展、行业趋势研判）
7. 整理本月政策/标准发布时间线
8. 参考 e:\trae_solo\自动化监测智能网联汽车新闻\templates\monthly_template.html 的设计风格生成HTML
9. 保存到 e:\trae_solo\自动化监测智能网联汽车新闻\reports\monthly\YYYY-MM.html

## 输出要求
- 深色主题数据仪表盘风格
- 包含数据概览统计卡片（总数+各类别数量，带count-up动画）
- SVG环形图展示分类占比
- SVG水平条形图展示来源分布Top 8
- "本月十大重要新闻"板块，前3名排名号高亮金色
- 六大分类各自的新闻汇编
- "本月趋势分析"段落
- "本月政策/标准时间线"垂直时间线
```

---

## 手动创建步骤

1. 在 TRAE 中打开 **Plan** 面板
2. 点击 **新建任务**
3. 填入上述 **名称** 和 **Cron 表达式**
4. 将对应的 **Message 内容** 粘贴到任务描述中
5. 保存并启用
